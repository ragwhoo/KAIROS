Phase 1 — Architecture Overview

Next.js with API Routes on Vercel. Clean separation in code, single deployable.

Frontend Layer

Next.js App Router pages & components. Tailwind + shadcn/ui. Framer Motion for animations.

API Layer

Next.js Route Handlers. Clean service/repository pattern. Prisma ORM for database access.

Shared Layer

Common types, validation schemas (Zod), utilities. Imported by both frontend and API layers.
